1. 화면 분할 (Pane 관리) 시작 전 ctrl + b 누르기
화면을 여러 개로 쪼개서 쓰고 싶을 때 사용합니다. 

% (Shift + 5): 화면을 좌우로 나눕니다.

" (Shift + '): 화면을 위아래로 나눕니다.

o (영문자): 나뉜 화면들 사이를 순서대로 이동합니다.

x: 현재 칸을 닫습니다 (y/n 확인 후).

z (Zoom): 현재 칸을 전체 화면으로 키웠다가 다시 줄였다가 합니다. (매우 유용!)

2. 새 창 만들기 (Window 관리)
화면 분할로도 부족할 때, 브라우저 탭처럼 새로운 창을 만듭니다.

c (Create): 새로운 빈 창을 생성합니다.

n (Next) / p (Previous): 다음 창 또는 이전 창으로 이동합니다.

0 ~ 9: 해당 번호의 창으로 바로 이동합니다.

w (Window list): 현재 열린 모든 창 목록을 보고 선택합니다.

3. 세션 관리 (강력 추천!)
Tmux의 꽃입니다. 아예 작업실 자체를 나갔다 들어오는 기능입니다.

d (Detach): 현재 Tmux 세션을 끄지 않고 밖(일반 터미널)으로 나갑니다.

(밖에서) tmux attach: 나갔던 작업실로 다시 복귀합니다.

s (Sessions): 현재 돌아가고 있는 전체 세션 목록을 봅니다.

-------------------------------
 세션 관리 (쉘 환경에서 입력)
새 세션 생성: tmux new -s <세션이름>
세션 목록 확인: tmux ls
세션 접속: tmux a -t <세션이름>
세션 종료 (삭제): tmux kill-session -t <세션이름>
세션 이름 변경: tmux rename-session -t <기존이름> <새이름> 

------------------------------------
tmux 내부 복사 붙여넣기
Ctrl + b를 누른 뒤 바로 [ 를 누릅니다.
그러면 copy mode로 들어갑니다.
방향키로 복사 시작 위치로 이동합니다.
Space를 눌러 시작점을 지정합니다.
방향키로 끝 위치까지 이동합니다.
Enter를 누르면 tmux 내부 버퍼에 복사됩니다.
붙여넣을 때는 Ctrl + b 다음 ] 를 누릅니다.
