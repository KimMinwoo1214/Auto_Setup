# Auto Setup

A collection of scripts for automatic computer setup.

## (Important) Requirements
What we only need is curl. Install curl before you run the command.

```bash
sudo apt-get update && sudo apt-get install curl git wget lsb-release -y
```

## How to use

Clone this repository and run the setup script:

```bash
git clone https://github.com/KimMinwoo1214/Auto_Setup.git
cd Auto_Setup
chmod +x *.sh
./all_install.sh
```

## Features

This setup includes several scripts for:
- Base utilities (git, wget, vim, etc.)
- Zsh and themes
- Chrome
- VSCode
- ROS2 Humble
- Miniconda
- LunarVim
- And more...
