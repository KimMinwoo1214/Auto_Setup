1. 파일 탐색기 열기 (NvimTree)

탐색기 열기/닫기: Space + e (Explorer의 약자)

탐색기 안에서 이동: j (아래), k (위)

폴더 열기/파일 선택: Enter

새 파일 만들기: 탐색기에서 a를 누르고 파일명 입력

파일 삭제: d를 누르고 y로 확인

파일명 변경: r (Rename)

2. 파일 및 텍스트 검색 (Telescope)

전체 프로젝트 파일 검색: Space + f (Find File)

창이 뜨면 파일 이름을 타이핑하고 Enter를 누르면 바로 열립니다.

코드 내 단어 검색 (Grep): Space + t (Text Search)

프로젝트 전체 파일에서 특정 단어가 포함된 줄을 모두 찾아줍니다.

최근 열었던 파일 검색: Space + r (Recent Files)

3. 화면 및 탭 관리 (Buffers)
여러 파일을 열었을 때 관리하는 방법입니다.

다음 탭으로 이동: L (대문자 L)

이전 탭으로 이동: H (대문자 H)

현재 탭 닫기: Space + c (Close)

4. 코드 편집의 기본 (Vim 모션)

편집 시작: i (Insert)

명령어 모드로 복귀: Esc (필수!)

저장하기: :w + Enter (Write)

종료하기: :q + Enter (Quit)

강제 종료: :q! + Enter
